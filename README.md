# Taobao Search Image Bot


##Details
+ 根据淘宝搜索页抓取商品详情中的图片
+ 可以一次执行多个搜索页的抓取任务
+ 已处理反爬虫机制
+ Windows 与 Linux 都可以执行 Python 2.7
+ 可以控制抓取数量
+ 抓取阶段实时监测
+ 分类抓取图片
+ 自动筛选装饰性图片
+ 支持天猫、淘宝、聚划算页面的图片抓取

例如 ：

抓取https://s.taobao.com/search?spm=a230r.1.14.39.s0V1z4&type=similar&app=i2i&rec_type=1&uniqpid=-972372134&nid=529793442152&s=60
中的内容还有其内部商品的图片。
